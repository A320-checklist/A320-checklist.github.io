A320neo Checklist

Cockpit Preparation

  • Gear Pins and Covers REMOVED
  • Fuel Quantity CHECKED
  • Batteries (check voltage > 25.5V) AUTO
  • ADIRUs 1, 2, 3 NAV
  • Signs ON/AUTO
  • BARO REF SET (BOTH)

Before Start

  • Parking Brake ON
  • Windows & Doors CLOSED
  • Beacon ON
  • Fuel Pumps ON
  • Thrust Levers IDLE
  • APU MASTER SW & BLEED ON

After Start

  • Dual Cooling (PW Engine) ON
  • ENG MODE Selector IGN/START
  • Engine Master 2 & 1 ON
  • APU BLEED & MASTER SW OFF
  • Ground Spoilers ARM
  • Pitch Trim & Rudder Trim SET / ZERO
  • Flaps SET FOR T/O
  • ECAM STATUS CHECKED
  • Anti-Ice AS REQ'D

Taxi

  • Flight Controls CHECKED
  • Flap Setting CONFIRM
  • ECAM MEMO (T.O.) CHECK (NO BLUE)

Before Takeoff (Line-up)

  • T.O Runway CONFIRM
  • RADAR & PRED W/S ON
  • Cabin Crew ADVISED
  • TCAS TA/RA
  • PACKS 1 & 2 AS REQ'D

After Takeoff / Climb

  • Landing Gear UP
  • Flaps RETRACTED
  • Ground Spoilers DISARMED
  • PACKS ON
  • Exterior Lights AS REQ'D
  • Seat Belts Sign OFF

Cruise

  • System Pages MONITOR
  • Fuel Prediction MONITOR

Approach

  • Descent Preparation COMPLETE
  • BARO REF SET (BOTH)
  • Seat Belts ON
  • Minimums SET (BOTH)
  • ENG MODE SEL AS REQ'D
  • G/A Altitude SET (BOTH)
  • Cabin Crew ADVISED
  • ECAM MEMO (LDG) CHECK (NO BLUE)

After Landing

  • Spoilers DISARMED
  • Flaps RETRACTED
  • RADAR & PRED W/S OFF
  • APU AS REQ'D
  • Exterior Lights AS REQ'D

Parking

  • Parking Brake or Chocks SET
  • Engines OFF
  • Beacon OFF
  • Fuel Pumps OFF

Securing The Aircraft

  • ADIRUs OFF
  • EMER EXIT LT OFF
  • Signs OFF
  • Batteries 1 & 2 OFF
